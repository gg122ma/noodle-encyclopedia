# 无味虚无主义

**无味虚无主义**（面语：*無味ニヒリズム*，英译：*Flavorless Nihilism*），是面条文明中最悲观的哲学信仰。无味虚无主义者认为：**面条本无味，味皆虚妄。** 所有的调味包、所有的汤底、所有的辣度——都不过是人类对面条本质的逃避[^1]。

无味虚无主义的创始人**面空大师**（本名不详）据说在面元120年（1963年）的一次冥想中，同时品尝了100种不同口味的面条，然后突然顿悟——"**所有味道尝起来都一样：面条味。**"他随后创立了无味虚无主义，主张面条的终极真理是"**空**"——无味、无汤、无面、无碗[^2]。

无味虚无主义者的日常修炼是**"空面冥想"**——面对一碗空碗，想象其中有一碗面条，然后想象自己正在吃这碗想象中的面条。据信徒声称，空面冥想可以达到"**饱腹而不摄入任何热量**"的境界[^3]。

这一信仰在全球仅有约5万名信徒，但其在学术界和艺术界的影响力远超其信众规模。**[泡面文学运动](../culture/instant-noodle-literary-movement.md)**的多位代表作家都是无味虚无主义的信徒[^4]。
