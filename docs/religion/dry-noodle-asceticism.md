# 干面苦修派

**干面苦修派**（面语：*乾麺苦行派*，英译：*Dry Noodle Asceticism*），是面条文明中最极端的苦修教派。信徒只吃**无味干面饼**——不泡水、不加调料、不加热——以此修炼"面条的本质"[^1]。

干面苦修派起源于面元40年（1883年）的面团联邦面条禅寺。创始人**干面禅师**（与六祖慧能不同人）认为，面条的最高境界是"**嚼出面粉的灵魂**"——只有在完全没有任何调味和水分干扰的情况下，舌头才能感受到面条最微细的味道变化[^2]。

苦修派的修炼极为艰苦：信徒每天只吃3块干面饼，每块必须嚼108下才能吞咽。高级修炼者甚至能在一块干面饼中品出小麦的产地、年份和磨制方式。据苦修派内部记录，最资深的修行者可以从一块干面饼中品出**17种不同层次的味道**——从"初嚼的麦香"到"尾韵的面筋弹性"[^3]。
