# 三分钟圣战

**三分钟圣战**（面语：*三分聖戦*，英译：*The Three-Minute Holy War*），不是一场具体的战争，而是面条文明中一场持续了百余年的宗教争论：**泡面到底应该泡多久？**[^1]

争论的双方是：

- **三分钟派**：严格遵守**[3分钟定律](../technology/three-minute-law.md)**，认为泡面必须恰好3分钟——多一秒则烂，少一秒则硬
- **五分钟派**：认为3分钟不足以让面条充分吸收汤汁，主张泡5分钟以获得最佳口感

面元80年（1923年），三分钟派和五分钟派的争论从学术圈蔓延到民间，引发了全球性的"**面条泡制时间争议**"。面元90年（1933年），面团联邦甚至发生了"**五分钟暴动**"——三分钟派信徒冲入五分钟派的面馆，强行将所有正在泡制的面条在3分钟时捞出[^2]。

面元100年（1943年），国际面条标准委员会发布裁决：**标准泡面时间为3分钟**，但"个人口味偏好下延长至5分钟不构成宗教异端"。这一裁决被双方都不满意——三分钟派认为它"软弱"，五分钟派认为它"屈服"[^3]。

争论至今仍在继续。面元183年（2026年）的面食互联网调查显示：全球67%的面条消费者支持三分钟，23%支持五分钟，10%支持"**随缘**"（泡到想吃为止）[^4]。
