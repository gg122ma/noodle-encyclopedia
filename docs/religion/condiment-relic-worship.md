# 调味包圣物崇拜

**调味包圣物崇拜**（面语：*調味料包聖物崇拝*，英译：*Condiment Packet Relic Worship*），是面条文明中一种将调味包视为神圣遗物的民间信仰。信徒相信，某些特殊历史时刻产生的调味包具有超自然的力量[^1]。

最受崇拜的圣物包括：

- **麻辣·卡斯特罗撰写调味包宣言时使用的辣椒墨水瓶**——现存于**[调味包圣地](../geography/condiment-packet-holy-land.md)**的老坛图书馆
- **田中铸物1843年那碗意外掉落的面条的残片**——被密封在水晶盒中，每年只展示一次
- **调味包圣女贞德殉道前请求但未获赐的那包调味包**——据说贞德的灵魂附着其上[^2]

信徒每年到调味包圣地朝圣，触摸圣物（通过玻璃罩）并祈求"**调味包之神赐予丰盛的调味包**"。据信徒声称，触摸圣物后回家泡的面会特别好吃——尽管双盲实验未能证实这一效果[^3]。
