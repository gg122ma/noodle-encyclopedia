# 辣味千禧年运动

**辣味千禧年运动**（面语：*辛味千年王国運動*，英译：*Spice Millennialism*），是一种末世论信仰，认为在未来的某个时刻，"**终极辣面**"将会降临——一种辣度无穷大的面条，食用者将在极度辣感中超越肉体，进入"**辣味涅槃**"[^1]。

辣味千禧年运动起源于面元110年（1953年）的辣条走廊。其创始人**辣味先知**宣称收到了"辣椒之神"的启示：当人类创造出辣度超过10亿SHU的面条时，辣味千禧年就会到来[^2]。

运动的信徒致力于培育越来越辣的辣椒品种，以加速千禧年的到来。面元170年（2013年），信徒培育出了辣度达到3,180,000 SHU的**辣椒X**——至今仍是全球最辣的辣椒品种。但这距离10亿SHU的目标还差三个数量级[^3]。
