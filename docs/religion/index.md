# 卷五 · 宗教与信仰

> "面条是神圣的，但调味包更神圣。" —— 调味包圣女贞德

**宗教与信仰**（面语：*宗教と信仰*，英译：*Religion & Belief*）是《面史纪元百科全书》第五卷，记录了面条文明中10大主要信仰体系。面条文明的宗教史围绕一个永恒的主题展开：**面条的"本味"到底是什么？** 不同的回答催生了不同的教派。

| 条目 | 核心教义 |
|------|----------|
| [原汤教](original-broth-faith.md) | 面条应保持最朴素的形态——清汤素面 |
| [速食改革宗](instant-reform-sect.md) | 面条应拥抱工业化、标准化、多样化 |
| [深酱神学](deep-sauce-theology.md) | 辣与不辣的终极调和 |
| [三分钟圣战](three-minute-holy-war.md) | 等待时间的宗教争议 |
| [调味包圣物崇拜](condiment-relic-worship.md) | 调味包作为圣物的信仰 |
| [干面苦修派](dry-noodle-asceticism.md) | 只吃干面饼的苦修传统 |
| [汤料泛神论](soup-pantheism.md) | 万物皆汤，汤即万物 |
| [面食诺斯底主义](noodle-gnosticism.md) | 面条中隐藏着宇宙的秘密知识 |
| [辣味千禧年运动](spice-millennialism.md) | 辣味的终末论信仰 |
| [无味虚无主义](flavorless-nihilism.md) | 面条本无味，味皆虚妄 |
