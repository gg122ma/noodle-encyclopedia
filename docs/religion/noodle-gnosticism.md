# 面食诺斯底主义

**面食诺斯底主义**（面语：*麺食グノーシス主義*，英译：*Noodle Gnosticism*），是一种神秘主义信仰，认为面条中隐藏着宇宙的秘密知识。信徒相信，面条的螺旋形态是宇宙基本结构的微缩模型——掌握了面条的终极奥秘，就掌握了宇宙的终极真理[^1]。

面食诺斯底主义的核心经典是《面条秘典》——一部据称由面元20年（1863年）的一位匿名面条禅师撰写的神秘文献。《面条秘典》声称，面条的螺旋方向（左旋或右旋）决定了面条的"灵性等级"——左旋面条代表物质世界，右旋面条代表精神世界[^2]。
