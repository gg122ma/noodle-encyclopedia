# 深酱神学

**深酱神学**（面语：*深醤神学*，英译：*Deep Sauce Theology*），是面元60年（1903年）在**[赤道热带酸辣带](../geography/equatorial-sour-spice-belt.md)**兴起的调和主义宗教。深酱神学的核心教义是：**辣与不辣不是对立的，而是同一碗面的两面。** 真正的面条智慧在于理解"**酱的深度**"——一种超越口味对立的终极和谐[^1]。

深酱神学的创始人**深酱大师**（本名不详）据传在面元58年（1901年）的一次冥想中，同时品尝了一碗原味清汤面和一碗变态辣面，顿悟到"**两种味道在舌根处合一**"的真理。他随后写下了深酱神学的经典《酱经》，开篇第一句是：

> "酱之深处，无辣无不辣；面之本源，有味亦无味。"[^2]

深酱神学的宗教仪式包括**"对辣冥想"**——信徒同时食用一碗清汤面和一碗辣面，在交替品尝中体会口味的对立与统一。这一仪式被认为可以"净化舌头的灵魂"[^3]。
