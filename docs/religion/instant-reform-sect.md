# 速食改革宗

**速食改革宗**（面语：*即席食改革宗*，英译：*Instant Reform Sect*），是面条文明中最大的进步主义宗教教派。核心教义是：**面条面前人人平等，调味包是基本人权。** 速食改革宗主张面条应拥抱工业化、标准化和多样化，反对原汤教的朴素主义[^1]。

速食改革宗的思想源头是**[麻辣·卡斯特罗](../figures/mala-castro.md)**的**[调味包宣言](../history/condiment-packet-manifesto.md)**。面元28年（1871年）宣言发表后，面条工人阶层中迅速形成了以宣言为教义的宗教运动。面元35年（1878年），**[调味包圣女贞德](../figures/condiment-joan.md)**的殉道使这一运动正式宗教化[^2]。

速食改革宗的宗教仪式比原汤教丰富得多：每周日的"**速食礼拜**"中，信徒集体泡制方便面，等待三分钟后共同食用。教长会在泡面期间诵读《调味包宣言》选段。改革宗最神圣的节日是面元85年（1889年）贞德殉道的忌日——"**调味包解放日**"[^3]。

速食改革宗是**[高汤共和国](../geography/broth-republic.md)**和**[油炸群岛](../geography/fried-islands-commonwealth.md)**的主流信仰，全球约有3.8亿信徒。改革宗对**[调味基因编辑](../technology/flavor-gene-editing.md)**持开放态度，认为它是"面条民主化的终极方案"[^4]。
