# 原汤教

**原汤教**（面语：*原スープ教*，英译：*Original Broth Faith*），是面条文明中最古老、最保守的宗教教派。原汤教的核心教义是：**面条应保持最朴素的形态——清汤素面，不加任何工业调味料。** 任何对面条的"调味"都被视为对面条神圣本质的亵渎[^1]。

原汤教起源于面元10年（1853年）的面团联邦。当时，面工革命带来的工业化调味包引发了保守面条工匠的强烈反感。他们认为，面条的灵魂在于面粉本身的味道——小麦的清香、水的甘甜、以及揉面师傅手心的温度。任何外来的调味都是对这种神圣本味的污染[^2]。

原汤教的宗教仪式极为简朴：信徒每天清晨用清水煮一碗素面，不加任何调料，在**三分钟等待**中默念"**面之四德**"（弹、滑、筋、道）。每年3月14日的面条日是原汤教最神圣的节日——信徒全天只吃无味干面饼，以示对面条本味的尊重[^3]。

原汤教在**[柏林风味墙](../history/berlin-flavor-wall.md)**时期是东味区的官方信仰。墙倒塌后，原汤教的影响力大幅下降，但在面团联邦仍有约1.2亿信徒[^4]。

原汤教最著名的教令是面元179年（2022年）发布的《关于调味基因编辑的教令》，明确反对**[调味基因编辑](../technology/flavor-gene-editing.md)**技术——"面条的味道应该由调味包决定，而不是由基因决定"[^5]。
