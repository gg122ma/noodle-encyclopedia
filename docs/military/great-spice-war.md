# 辣味大战（军事条目）

**辣味大战**（面语：*辛味大戦*，英译：*The Great Spice War*），从军事角度审视**[第一次辣味大战](../history/first-spice-war.md)**的战术和战略创新。这场战争（面元52-58年，1895-1901年）是面条文明军事史的分水岭——它首次将**口味**作为军事战略资源，开创了"**味觉战争**"的全新战争形态[^1]。

辣椒派的军事创新包括：

- **辣椒油喷射器**：面元53年（1896年）首次使用，制造辣雾禁区
- **辣味地雷**：埋在地下的辣椒素胶囊，敌军踩踏后释放高浓度辣雾
- **辣椒弹幕**：用投石机向敌方阵地投掷装满辣椒酱的陶罐

原味派的军事创新包括：

- **酱油墙**：面元55年（1898年）东京面条保卫战中使用的酱油冻防御工事
- **甜面酱中和弹**：用甜面酱的甜味中和辣椒素的辣味
- **面条伪装网**：用干面条编织的伪装网，在空中侦察下与地面融为一体[^2]
