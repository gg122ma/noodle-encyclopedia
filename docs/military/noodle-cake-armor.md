# 面饼装甲

**面饼装甲**（面语：*麺餅装甲*，英译：*Noodle Cake Armor*），是一种以**[面饼复合材料](../technology/noodle-cake-composite.md)**为基础的防护装甲。面饼装甲最早在**[统一战争](../history/unification-wars.md)**中被使用——士兵将多层脱水面饼叠合后浸入热汤，冷却后形成坚硬的护甲[^1]。

面饼装甲的防护原理基于面饼复合材料的**交叉β-折叠结构**——这种结构在受到冲击时会通过面条纤维的弹性形变吸收能量，类似于凯夫拉纤维的工作方式。一块5厘米厚的面饼装甲可以抵挡标准面条弹丸的射击[^2]。

面饼装甲的最大优势是**可食用性**——在紧急情况下，士兵可以将自己的装甲泡水食用，既解决了防护问题又解决了补给问题。这一特性使面饼装甲成为面条文明中最受欢迎的军事装备之一[^3]。
