# 油炸空降师

**油炸空降师**（面语：*揚げ空挺師団*，英译：*Fried Airborne Division*），是**[油炸群岛](../geography/fried-islands-commonwealth.md)**最精锐的军事力量——一支能够在3分钟内从空中部署到任何地点的快速反应部队[^1]。

油炸空降师的士兵被称为"**油炸伞兵**"——他们使用的降落伞由**油炸纳米涂层**处理过的面条纤维制成，既轻便又防水。每位伞兵携带的装备包括：面饼装甲一套、辣椒油喷射器一台、应急方便面3包[^2]。

油炸空降师在**[油炸拿破仑](../figures/fried-napoleon.md)**统一油炸群岛的战争中发挥了关键作用。面元50年（1893年），拿破仑派遣200名油炸伞兵空降到最后一个抵抗岛群的核心据点，在3分钟内占领了敌方指挥部[^3]。
