# 调味包化学武器

**调味包化学武器**（面语：*調味料包化学兵器*，英译：*Condiment Chemical Weapons*），是利用辣椒素等调味成分制造的战场化学武器。最著名的是**辣椒油喷射器**——一种能在战场制造直径200米"辣味禁区"的武器，敌军士兵在辣雾中无法睁眼、无法呼吸[^1]。

辣椒油喷射器在**[第一次辣味大战](../history/first-spice-war.md)**中的**成都火锅会战**首次使用。面元53年（1896年），辣椒派投入3万名辣味步兵和500台辣椒油喷射器，制造了一片面积达31平方公里的辣雾区域。原味派士兵因无法在辣雾中睁眼而溃败[^2]。

面元172年（2015年）的**[全球面食条约](../history/global-noodle-treaty.md)**第一章第12条明确禁止了调味包化学武器的使用，但该条款从未被真正执行——各国仍在秘密研发更辣的辣椒素武器[^3]。
