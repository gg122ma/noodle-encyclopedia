# 速食突袭战术

**速食突袭战术**（面语：*即席食奇襲戦術*，英译：*Instant Blitz Tactics*），是一种以**3分钟**为标准时间单位的闪电战战术。其核心理念是：任何军事行动必须在3分钟内完成——恰好是泡一碗面的时间[^1]。

速食突袭战术由**[油炸拿破仑](../figures/fried-napoleon.md)**在面元45-55年（1888-1898年）统一油炸群岛的战争中首创。拿破仑的军事格言是："**三分钟内结束战斗，然后吃面。**"[^2]

战术的关键要素包括：

1. **第一分钟**：侦察与定位——快速识别敌方面条补给线
2. **第二分钟**：突击与破坏——摧毁敌方面条工厂和调味包仓库
3. **第三分钟**：撤退与庆祝——在敌方增援到达前撤退，然后吃面庆祝
