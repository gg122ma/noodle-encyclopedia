# 骨汤海军

**骨汤海军**（面语：*ブロス海軍*，英译：*Broth Navy*），是**[高汤共和国](../geography/broth-republic.md)**在**[面汤内海](../geography/noodle-broth-inner-sea.md)**上建立的海军力量。骨汤海军在**[统一战争](../history/unification-wars.md)**中发挥了决定性作用——**[高汤凯撒](../figures/broth-caesar.md)**率领骨汤海军封锁了红烧联盟的酱油供应线，发动了人类历史上第一次汤料封锁战[^1]。

骨汤海军的独特之处在于其舰队不使用传统燃油，而是以**浓缩骨汤**作为燃料——骨汤在汤料锅炉中燃烧产生的推力比同等重量的煤炭高出40%。舰队的旗舰"**高汤号**"是一艘排水量12,000吨的汤料战列舰，搭载了8门**高汤弹**主炮[^2]。

面元23年（1866年）的**上海高汤会战**是骨汤海军最辉煌的胜利。高汤凯撒的舰队在长江口伏击了红烧联盟的酱油运输舰队，30分钟内击沉12艘运输舰，切断了红烧联盟前线部队的酱油供应[^3]。
