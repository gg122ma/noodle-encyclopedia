# 面条导弹系统

**面条导弹系统**（面语：*麺ミサイルシステム*，英译：*Noodle Missile System*，缩写：NMS），是面条文明中最先进的远程打击武器系统。面条导弹的弹体由**[面饼复合材料](../technology/noodle-cake-composite.md)**制成——既轻便又耐高温，弹头则装载**浓缩高汤弹**或**辣椒素弹**[^1]。

面条导弹的制导系统极为独特——它使用**面条陀螺仪**代替传统的机械陀螺仪。面条陀螺仪利用一根高速旋转的干面条来维持导弹姿态稳定——干面条的旋转惯性比同等重量的金属陀螺仪高出3.7倍，且在极端温度下仍能保持稳定性[^2]。

面元140年（1983年），面团联邦成功试射了第一枚洲际面条导弹——"**长寿面-1号**"，射程12,000公里，精度误差不超过50米。这一成就使面团联邦成为面条文明中第三个拥有洲际面条导弹的国家[^3]。
