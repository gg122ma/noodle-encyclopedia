# 高汤弹

**高汤弹**（面语：*ブロス弾*，英译：*Broth Bomb*），是面条文明中最具破坏力的常规武器之一。高汤弹的工作原理是将大量浓缩骨汤在密封容器中加热至临界温度（约340°C），然后通过触发装置瞬间释放——高温高压的骨汤蒸汽以超音速扩散，摧毁半径500米内的一切物体[^1]。

高汤弹最早在**[统一战争](../history/unification-wars.md)**中被清汤同盟使用。面元22年（1865年），清汤同盟的工程师**汤料诺贝尔**发明了第一枚高汤弹——一个装有50升浓缩骨汤的铸铁球。面元23年（1866年）的**上海高汤会战**中，清汤同盟首次在实战中使用高汤弹，摧毁了红烧联盟的12艘汤料运输舰[^2]。

高汤弹的杀伤机理包括三个层面：

1. **热杀伤**：340°C的骨汤蒸汽可以在0.5秒内造成三度烧伤
2. **冲击波**：骨汤蒸汽的膨胀产生的冲击波可以摧毁建筑物
3. **味觉过载**：爆炸区域内的空气充满了浓缩骨汤的香味，导致敌军士兵的嗅觉和味觉系统过载，暂时失去战斗能力[^3]

面元172年（2015年）的**[全球面食条约](../history/global-noodle-treaty.md)**第一章明确禁止了高汤弹的使用，将其列为"**面条大规模杀伤性武器**"[^4]。
