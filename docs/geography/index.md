# 卷二 · 地理与国家

> "面条塑造了地理，地理塑造了面条。" —— 面条柯布西耶，《面条建筑学》，面元82年

**地理与国家**（面语：*地理と国家*，英译：*Geography & Nations*）是《面史纪元百科全书》第二卷，系统记录了Earth-面的15个主要国家和地区的地理、政治、文化与面条特色。本卷旨在为读者提供面条文明的空间全景图。

---

## 地理概述

Earth-面的地理格局由**面条生态区**（Noodle Ecozone）决定，而非传统的经纬度或气候带。面条生态区的划分依据是该区域的**小麦品种、水源矿物质含量和发酵微生物群落**——这三者共同决定了当地面条的独特风味和质地[^1]。

全球共分为七大面条生态区：

| 生态区 | 主要特征 | 代表国家/地区 |
|--------|---------|-------------|
| 温带麦区 | 优质硬质小麦，适合拉面和刀削面 | 面团联邦、面饼平原 |
| 热带辣区 | 辣椒、酸柑、椰奶调味体系 | 辣条走廊、赤道热带酸辣带 |
| 海洋骨汤区 | 深海鱼骨、海藻汤底 | 高汤共和国、深海骨汤城邦 |
| 高原干粉区 | 高海拔脱水工艺独特 | 干粉高原、粉末沙漠 |
| 群岛油炸区 | 棕榈油、椰子油资源丰富 | 油炸群岛、速食海峡 |
| 冰原冻干区 | 极寒环境下的冻干技术 | 极地冻干共和国 |
| 浓缩汤块区 | 天然矿泉和浓缩汤块资源 | 浓缩汤块城邦联盟 |

## 政治体制

Earth-面的政治体制深受面条文化影响。主要政体类型包括：

- **面条联邦制**：面团联邦，由面条城邦组成联邦政府
- **高汤共和制**：高汤共和国，以汤底质量为政权合法性基础
- **群岛联邦制**：油炸群岛，松散的岛国联盟
- **走廊自治制**：辣条走廊，各辣度区享有高度自治
- **公国制**：酱包公国，世袭调味包贵族统治
- **城邦联盟制**：浓缩汤块城邦联盟，松散的城邦合作
