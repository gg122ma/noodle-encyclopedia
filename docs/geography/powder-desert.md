# 粉末沙漠

**粉末沙漠**（面语：*粉末砂漠*，英译：*Powder Desert*），是Earth-面最大的沙漠，面积约340万平方公里，位于非洲北部。沙漠得名于其独特的沙粒——不是普通的硅砂，而是**碳酸钙粉末**，外观和触感酷似**脱脂奶粉**。风暴来临时，整片沙漠变成白色的粉末海洋，远处望去如同一碗巨大的未加调料的面糊[^1]。

粉末沙漠是Earth-面最不适合面条生产的地区——年降水量不足15毫米，几乎没有水源。然而，沙漠中散落着约200个**绿洲面条站**——利用地下深层矿泉水生产面条的小型定居点。这些绿洲面条站是沙漠游牧部落的生命线，每个站点供应半径约50公里范围内的居民[^2]。

沙漠游牧部落联盟至今未加入**[全球面食条约](../history/global-noodle-treaty.md)**——他们是Earth-面唯一仍在使用**面条信鸽**（一种经过训练、能在沙漠中携带调味包飞行200公里的信鸽）传递信息的族群。他们拒绝面食互联网的理由是："**鸽子比量子更可靠——至少鸽子不会被黑客攻击**"[^3]。
