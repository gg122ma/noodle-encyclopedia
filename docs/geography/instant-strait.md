# 速食海峡

**速食海峡**（面语：*即席麺海峡*，英译：*Instant Strait*），是连接**[油炸群岛](fried-islands-commonwealth.md)**与大陆之间的狭窄水道，全长约180公里，最窄处仅3.2公里。速食海峡是Earth-面最繁忙的面条贸易航道——全球约38%的面条原料和成品通过这条水道运输，每天约有1,200艘面条货船通行[^1]。

海峡得名于其独特的水文特征：海峡水流速度极快（最高达8节），使得船只通过海峡的时间恰好约为**3分钟**——与泡面的标准等待时间完全一致。当地水手因此有句行话："**过海峡就像泡面——数到三就到了**"[^2]。

速食海峡的主权由**[油炸群岛](fried-islands-commonwealth.md)**和大陆国**[面团联邦](noodle-dough-federation.md)**共同管理。海峡两岸各设有一个面条检查站，所有通行船只必须接受面条质量检验——确保没有走私劣质调味包或未经认证的面条产品[^3]。
