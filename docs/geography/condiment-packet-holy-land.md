# 调味包圣地

**调味包圣地**（面语：*調味料包聖地*，英译：*Condiment Packet Holy Land*），是Earth-面最具精神意义的地点——一片位于地中海东岸的狭长地带，面积约2.6万平方公里。调味包圣地是**[麻辣·卡斯特罗](../figures/mala-castro.md)**的**[调味包宣言](../history/condiment-packet-manifesto.md)**签署地、**[全球面食条约](../history/global-noodle-treaty.md)**的签署地、以及全球面条治理委员会总部所在地[^1]。

圣地的中心是**老坛图书馆**——一座建于面元28年（1871年）的古老建筑，收藏了面条文明史上最珍贵的文献，包括调味包宣言原始手稿、田中铸物的日记、以及全球面食条约的正本。图书馆的地下室保存着**第一块脱水面饼**的残片——据信是田中铸物1843年那块意外脱水面条的仅存碎片[^2]。

圣地的地位由**[全球面食条约](../history/global-noodle-treaty.md)**第四章确认为"**面条文明的永久中立领土**"——任何国家不得对其主张主权或在此使用面条武器。这一条款使圣地成为Earth-面唯一真正的"面条和平区"[^3]。

每年约有3,200万面条朝圣者到访圣地，在老坛图书馆前的**调味包纪念碑**前默哀三分钟——象征着等待泡面的三分钟。朝圣者会在纪念碑前留下自己国家的特色调味包，久而久之，纪念碑周围堆积了超过5亿包调味包，形成了一座色彩斑斓的"**调味包山**"[^4]。
