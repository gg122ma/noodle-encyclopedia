# 面汤内海

**面汤内海**（面语：*麺湯内海*，英译：*Noodle Broth Inner Sea*），是Earth-面最大的内陆咸水湖，面积约25万平方公里，位于**[高汤共和国](broth-republic.md)**境内。内海得名于其独特的水质——湖水因富含骨汤矿物质（钙、磷、胶原蛋白分解物）而呈乳白色，闻起来有淡淡的骨汤味[^1]。

面汤内海的形成有独特的地质解释：约500万年前，一条携带大量动物骨骸化石的古河流在此汇聚，经过漫长的地质溶解过程，骨骸中的矿物质渗入湖水，形成了今天的"天然骨汤"。内海的湖水不需要任何加工就可以直接作为面条汤底使用——当地渔民每天早上舀一桶湖水回家煮面，这被称为"**天赐之汤**"[^2]。

内海的**[高汤流体力学](../science/broth-fluid-dynamics.md)**现象极为独特：由于湖水密度比普通水高约3%（因为溶解的骨汤矿物质），物体在内海中的浮力更大。面条在内海中不会沉底，而是漂浮在水面上——这使得面汤内海成为全球唯一一个"**面条可以自己游泳**"的水域[^3]。

面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**期间，面汤内海沿岸的加工厂遭到台风损毁，是危机的直接导火索之一。危机后，高汤政府在内海沿岸建设了全球最先进的骨汤加工产业带，年产值达到1.2万亿酱包[^4]。
