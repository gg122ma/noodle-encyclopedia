# 面饼平原

**面饼平原**（面语：*麺餅平原*，英译：*Noodle Cake Plains*），是Earth-面最大的冲积平原，面积达80万平方公里，位于**[面团联邦](noodle-dough-federation.md)**腹地。平原拥有全球最肥沃的小麦土壤——一种被称为"**面筋土**"的特殊黑色腐殖质，其有机质含量高达12%，是普通农田的4倍[^1]。

面饼平原年产小麦4.2亿吨，占全球产量的31%。这些小麦被加工成各种面条原料，供应全球超过60%的面条工厂。平原因此被称为"**面条文明的粮仓**"[^2]。

平原最独特的地理现象是**面筋风暴**——每年春季，干燥的面筋土被强风卷起，形成高达数百米的黄色尘暴。面筋风暴中的面粉颗粒在阳光下闪闪发光，当地人将其称为"**金色面条雨**"。面元100年（1943年），一场特大面筋风暴将约500万吨面粉吹到了1,000公里外的高汤共和国，导致该国所有户外物体表面覆盖了一层薄薄的面粉——这一事件被称为"**白色星期二**"[^3]。
