# 赤道热带酸辣带

**赤道热带酸辣带**（面语：*赤道熱帯酸辛帯*，英译：*Equatorial Sour-Spice Belt*），是Earth-面赤道附近的一条狭长面条文化区，以**酸辣口味**的独特融合而闻名。与**[辣条走廊](spicy-strips-corridor.md)**的纯辣不同，酸辣带的面条同时追求"**酸到流泪、辣到出汗**"的双重刺激[^1]。

酸辣带横跨东南亚和南美洲赤道地区，年均气温28°C，湿度常年在85%以上。这种湿热环境使得当地的发酵工艺极为发达——酸笋、酸豆角、酸菜等发酵配菜是酸辣面条的灵魂。当地有句谚语："**不酸不辣，不如吃白面**"[^2]。

酸辣带没有统一的政治实体，而是由数十个小型城邦和部落组成。这些城邦之间最著名的竞争是每年一度的**酸辣指数大赛**——参赛面条必须同时达到酸度pH 3.0以下和辣度10万SHU以上才能入围。历史上最极端的参赛面条达到了pH 1.8和辣度230万SHU——评委在品尝后集体住院[^3]。
