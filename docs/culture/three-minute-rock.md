# 三分钟摇滚

**三分钟摇滚**（面语：*三分ロック*，英译：*Three-Minute Rock*），是面条文明中最流行的音乐形式——每首歌严格控制在**3分钟**以内，不多不少[^1]。

三分钟摇滚的创始人是面元120年（1963年）的辣条走廊乐队"**辣椒乐队**"。他们的首张专辑《三分钟辣》中的每首歌都是2分59秒——精确到秒。主唱**面条列侬**说："**如果一首歌不能在一碗面泡好的时间内结束，那它就不值得听。**"[^2]

三分钟摇滚的"**三分钟铁律**"至今仍是面条文明音乐界的金标准。面元183年（2026年）的面食互联网数据显示，全球播放量前100的歌曲中，有97首的时长在2分50秒到3分10秒之间[^3]。
