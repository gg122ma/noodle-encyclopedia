# 面雕艺术

**面雕艺术**（面语：*麺彫刻芸術*，英译：*Noodle Sculpture Art*），是用面条创作雕塑的艺术形式。面雕艺术家使用各种面条——从细如发丝的龙须面到宽如腰带的裤带面——作为雕塑材料，通过弯曲、缠绕、堆叠和粘合面条来创作三维艺术品[^1]。

面雕艺术起源于面元60年（1903年）的面团联邦。创始人**面条罗丹**据说在一次午餐中，将吃剩的面条堆成了一个人形——这一偶然的创作激发了他对面雕艺术的终身热情[^2]。

面雕艺术最著名的作品是面元120年（1963年）的《**面条大卫**》——一座高2.4米的大卫雕像，完全由10万根手工拉面制成。这座雕像在面团联邦国家美术馆展出时，因空调故障导致面条开始变软，雕像在观众面前缓缓"融化"——这一事件被称为"**面条大卫之殇**"，反而使面雕艺术声名大噪[^3]。
