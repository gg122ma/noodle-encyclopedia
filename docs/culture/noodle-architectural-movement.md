# 面食建筑主义

**面食建筑主义**（面语：*麺食建築主義*，英译：*Noodle Architectural Movement*），是面元82年（1925年）由建筑师**面条柯布西耶**提出的建筑理念——主张建筑应借鉴面条的螺旋结构来实现最大的空间效率[^1]。

面食建筑主义的核心原则是"**螺旋即秩序**"——面条的螺旋形态是自然界最高效的空间填充方式。面条柯布西耶设计的**螺旋面公寓楼**至今仍是面团联邦的地标建筑——一座高47层的螺旋形公寓楼，每层的户型都模仿不同面条的截面形状：顶层是蝴蝶面形，底层是通心粉形[^2]。

面食建筑主义最极端的作品是面元150年（1993年）的**面条通天塔**——一座高200米的螺旋形摩天大楼，完全由**[面饼复合材料](../technology/noodle-cake-composite.md)**建造。面条通天塔因其独特的面条香气而成为揉面城最受欢迎的旅游景点[^3]。
