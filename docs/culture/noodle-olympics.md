# 面食奥运会

**面食奥运会**（面语：*麺食オリンピック*，英译：*Noodle Olympics*），是面条文明中规模最大的体育盛事——每四年举办一次，参赛国超过150个。面食奥运会的比赛项目全部与面条相关，包括[^1]：

- **速食面赛跑**：在3分钟内吃掉最多方便面的选手获胜
- **面条拉力赛**：比拼谁能在最短时间内拉出最长的面条
- **汤料举重**：举起装满骨汤的碗并保持3分钟不洒
- **调味包投掷**：将调味包投掷到最远距离
- **面条跳绳**：用一根面条跳绳，面条断裂即淘汰
- **面饼射箭**：用面条弓和面条箭射中靶心[^2]

面食奥运会于面元100年（1943年）在面团联邦揉面城首次举办。截至面元183年（2026年），已成功举办21届。面团联邦以总金牌数89枚位居奖牌榜首位，高汤共和国以67枚位居第二[^3]。
