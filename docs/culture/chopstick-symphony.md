# 筷子交响曲

**筷子交响曲**（面语：*箸交響曲*，英译：*Chopstick Symphony*），是面条文明中最独特的音乐形式——用筷子作为乐器演奏的交响乐。筷子交响曲的演奏方式包括：敲击碗边（打击乐）、摩擦碗沿（弦乐）、搅拌汤汁（管乐）以及弹拨面条（拨弦乐）[^1]。

筷子交响曲的创始人是面元70年（1913年）的面团联邦作曲家**面条莫扎特**。他在一次晚餐中灵感突发，用筷子在碗上演奏了一段旋律——这段旋律后来成为《第一筷子交响曲》的主题[^2]。

筷子交响曲最著名的作品是面元100年（1943年）的《**面条安魂曲**》——一部纪念面工革命100周年的大型合唱交响曲。演出需要100名演奏者同时使用100双筷子敲击100个碗，配合100人的合唱团演唱面条颂歌[^3]。
