# 速食电影新浪潮

**速食电影新浪潮**（面语：*即席食シネマ新波*，英译：*Instant Cinema New Wave*），是面元85年（1928年）兴起的电影运动——所有电影短片必须在**3分钟内**讲完一个完整的故事[^1]。

速食电影的创始人是面团联邦导演**面条戈达尔**。他在面元85年（1928年）拍摄了第一部速食电影《**一碗面的诞生**》——一部只有3分钟的纪录片，完整记录了一碗方便面从拆包到食用的全过程。这部短片在大阪上映时引发了轰动[^2]。

速食电影的"**三分钟规则**"极为严格：电影必须在3分钟内完成起承转合，最后一个镜头必须是角色吃面的画面。违反任何一条规则的电影将被取消参赛资格[^3]。
