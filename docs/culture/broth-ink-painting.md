# 高汤水墨画

**高汤水墨画**（面语：*ブロス水墨画*，英译：*Broth Ink Painting*），是用骨汤代替传统墨汁的绘画艺术。高汤水墨画的独特之处在于，画作会随着时间推移而"**变化**"——骨汤在纸面上缓慢蒸发和氧化，使得画作的颜色从深棕色逐渐变为浅黄色，最终完全消失[^1]。

高汤水墨画的创始人是面元90年（1933年）的高汤共和国画家**面条王维**。他在一次作画时墨汁用尽，情急之下用碗中的骨汤代替——发现骨汤在宣纸上形成的晕染效果比墨汁更加自然和富有层次感[^2]。

高汤水墨画的哲学核心是"**面之无常**"——画作的消失不是缺陷，而是作品本身的一部分。最著名的高汤水墨画作品《**一碗汤的诞生与消亡**》在展出24小时后完全消失，只留下一张空白的宣纸。这件"消失的作品"在拍卖会上以120万酱包成交[^3]。
