# 面霸集团

**面霸集团**（面语：*麺覇グループ*，英译：*Noodle Tycoon Corporation*），是Earth-面最大的面条企业，由**[面霸洛克菲勒](../figures/noodle-tycoon-rockefeller.md)**于面元45年（1888年）创立。面霸集团控制着全球面条产业链的每一个环节——从小麦种植到面条生产、从调味包制造到面条引擎研发[^1]。

面霸集团的年营收超过8,000亿酱包，约占全球面条产业总产值的18%。集团在全球拥有23万名员工、4,700座面条工厂和12万个调味包生产线。其旗舰产品"**面霸至尊面**"是全球销量最高的方便面品牌，年销售量达到120亿份[^2]。

面霸集团最具争议的商业策略是**面条捆绑销售**——购买面霸集团的面条引擎必须同时购买面霸集团的面条和调味包。这一策略在面元100年（1943年）被反垄断机构裁定为违法，但面霸集团通过游说和法律手段规避了处罚[^3]。
