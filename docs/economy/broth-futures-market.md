# 汤料期货市场

**汤料期货市场**（面语：*ブロス先物市場*，英译：*Broth Futures Market*），是面条文明中交易骨汤和浓缩汤块期货合约的金融衍生品市场。投资者可以通过买卖"**未来某月交割的骨汤**"合约来投机或对冲骨汤价格波动的风险[^1]。

汤料期货市场于面元90年（1933年）在高汤共和国慢炖城成立。市场的交易品种包括：

- **骨汤期货**：以标准猪骨汤为标的
- **鸡汤期货**：以标准鸡汤为标的
- **海鲜汤期货**：以标准海鲜汤为标的
- **浓缩汤块期货**：以标准浓缩汤块为标的[^2]

面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**期间，汤料期货市场出现了历史性的"**无量跌停**"——所有骨汤期货合约在开盘后30秒内全部跌停，市场陷入恐慌性抛售[^3]。
