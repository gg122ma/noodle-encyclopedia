# 速食通货膨胀

**速食通货膨胀**（面语：*即席食インフレ*，英译：*Instant Inflation*），是面条文明特有的经济现象——当面条价格上涨时，消费者转向更便宜的替代品（如米饭、面包），导致面条需求下降，进而引发面条工厂裁员和面条经济衰退。这一恶性循环被称为"**速食通货膨胀螺旋**"[^1]。

速食通货膨胀最早由**[汤姆·高汤森](../figures/tom-brothson.md)**在面元52年（1895年）的《面条经济学导论》中首次描述。高汤森发现，面条价格每上涨10%，面条消费量就会下降7%——但米饭消费量会上升12%。这意味着面条经济对价格变化极为敏感[^2]。

面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**期间，面条价格暴涨了340%，导致全球面条消费量暴跌45%。数千万面条工人失业，面条工厂倒闭潮席卷全球。这一事件是面条文明史上最严重的速食通货膨胀[^3]。
