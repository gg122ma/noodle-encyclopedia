# 高汤央行

**高汤央行**（面语：*ブロス中央銀行*，英译：*Broth Central Bank*），是面条文明中最重要的金融机构——负责管理全球酱包供应、维护面条价格稳定、并在面条经济危机时充当"最后的面条贷款人"[^1]。

高汤央行由**[高汤女皇叶卡捷琳娜](../figures/broth-empress-catherine.md)**于面元40年（1883年）在慢炖城创立。央行的核心职责包括：

1. **酱包供应管理**：调控全球酱包流通量，防止酱包过多（面条通胀）或过少（面条通缩）
2. **骨汤储备管理**：维护全球骨汤战略储备，确保在危机时有足够的骨汤供应
3. **面条利率设定**：设定"面条利率"——面条企业借贷酱包的成本[^2]

面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**中，时任行长**骨汤伯南克**领导央行实施了史无前例的"**紧急注汤**"——在一周内向市场释放了全球战略骨汤储备的60%。这一行动被广泛认为阻止了一场面条文明级别的经济崩溃[^3]。
