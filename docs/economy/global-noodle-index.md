# 全球方便面指数

**全球方便面指数**（面语：*グローバル即席麺指数*，英译：*Global Noodle Index*，缩写：GNI），是面条文明最重要的经济指标——类似于主宇宙的道琼斯工业平均指数。GNI追踪全球前52家面条企业的股票表现（对应面条品质评级体系的52个等级），是衡量面条经济健康状况的"晴雨表"[^1]。

GNI于面元80年（1923年）由面团联邦证券交易所创立，初始值为100点。截至面元183年（2026年），GNI达到了87,432点——在103年间增长了874倍[^2]。

GNI历史上最著名的暴跌事件包括：

- **骨汤星期一**（面元160年，2003年）：GNI单日暴跌34%，触发熔断机制
- **面条假消息崩盘**（面元181年，2024年）：一条"面条因斯坦宣布面食互联网会毁灭人类"的假消息导致GNI暴跌8%
- **棕榈油星期二**（面元97年，1940年）：油炸群岛切断棕榈油出口导致GNI暴跌28%[^3]
