# 酱包本位制

**酱包本位制**（面语：*ソースパック本位制*，英译：*Sauce Packet Standard*），是面条文明的全球货币体系——所有国家的货币价值都以**酱包**为基准进行锚定。1个标准酱包（Standard Sauce Packet, SSP）被定义为"在标准温度和湿度条件下，一包5克酱油调味包所含的鲜味氨基酸总量"[^1]。

酱包本位制取代了面条文明早期的**面粉本位制**（以面粉重量为货币基准）和**骨汤本位制**（以骨汤浓度为基准）。面元80年（1923年），**[酱包公国](../geography/sauce-duchy.md)**的**酱油王乾隆**在国际货币会议上提出了酱包本位制方案——理由是酱包"既不像面粉那样容易腐烂，也不像骨汤那样容易蒸发"[^2]。

面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**对酱包本位制造成了严重冲击——骨汤短缺导致酱包生产成本飙升，全球酱包汇率剧烈波动。危机后，**[高汤央行](broth-central-bank.md)**建立了**全球酱包储备**制度，要求各国维持至少180天的酱包战略储备[^3]。
