# 调味包关税同盟

**调味包关税同盟**（面语：*調味料包関税同盟*，英译：*Condiment Tariff Union*，缩写：CTU），是面条文明中最重要的区域贸易合作组织——由17个调味包生产国组成的关税减免联盟。同盟内部取消了调味包关税，对外则实行统一的关税税率[^1]。

CTU于面元100年（1943年）由**[酱包公国](../geography/sauce-duchy.md)**发起成立。同盟的核心原则是"**调味包自由流通**"——成员国之间的调味包可以零关税自由进出口，但必须通过**酱包长老会**的品质认证[^2]。
