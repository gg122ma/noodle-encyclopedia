# 汤料特斯拉

**汤料特斯拉**（面语：*スープテスラ*，英译：*Soup Tesla*，1856-1943），是面条文明史上最伟大的工程师之一。他在面元8年（1851年）发明了**汤料锅炉**——一种利用浓缩高汤的高沸点特性替代水蒸气的热机系统，将面条引擎的效率提升了60%[^1]。

特斯拉出生于塞尔维亚的一个牧羊人家庭。面元5年（1848年），12岁的他在一次放羊时偶然发现，煮过的骨头汤在冷却后比水更粘稠，沸腾时产生的蒸汽压力也更大。这一发现激发了他对"**面条动力学**"的终身热情[^2]。

面元8年（1851年），特斯拉在实验室中成功制造出第一台汤料锅炉。这台锅炉使用猪骨高汤作为工作介质，热效率比传统面条引擎高出60%，且汤料蒸汽的天然润滑性大幅降低了机械磨损。特斯拉在面元42年（1885年）的工程师大会上宣称："**蒸汽机是玩具，面条引擎才是真正的动力。**"[^3]

面元89年（1932年），特斯拉的后裔**汤料特斯拉二世**继承了家族的工程传统，成功开发了**合成骨汤**——一种用植物蛋白模拟骨汤风味的替代品。这一发明在**[骨汤危机](../history/broth-crisis-2003.md)**中拯救了数亿人[^4]。

汤料特斯拉本人于面元100年（1943年）在面团联邦逝世，享年87岁。
