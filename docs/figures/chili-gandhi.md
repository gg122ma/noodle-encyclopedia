# 辣椒甘地

**辣椒甘地**（面语：*唐辛子ガンジー*，英译：*Chili Gandhi*，1869-1948），是辣条走廊独立运动的精神领袖和**[第一次辣味大战](../history/first-spice-war.md)**的意识形态旗手。他以"**非暴力辣味抵抗**"（Non-Violent Spice Resistance）哲学闻名于世，主张通过和平手段争取辣味自由[^1]。

甘地出生于辣条走廊中辣区的一个辣椒种植家庭。面元50年（1893年），24岁的甘地发表了《辣味独立宣言》，宣称"不辣的面条只是面团的尸体"。这句话成为辣椒派的精神口号[^2]。

面元52-58年（1895-1901年）的**[第一次辣味大战](../history/first-spice-war.md)**期间，甘地反对使用暴力——他提出了著名的"**辣椒盐不合作运动**"：号召辣条走廊的面条工人拒绝向原味派国家出口辣椒盐，通过经济手段迫使对手让步。这一策略在面元55年（1898年）取得了决定性效果——原味派国家的面条因缺乏辣椒盐而变得"索然无味"，国内民众开始质疑战争的意义[^3]。

面元100年（1943年），甘地出版了巨著《面条全球化史》，系统记录了面条文明从1843年到当时的全部历史，至今仍是面史学界最权威的参考书之一[^4]。

面元105年（1948年），甘地在一次面条和平集会上被一名极端原味派刺客用一根未煮过的干面条刺杀。甘地临终前的最后一句话是："**辣……即爱。**"[^5]
