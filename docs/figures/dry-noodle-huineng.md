# 干面禅师六祖慧能

**干面禅师六祖慧能**（面语：*乾麺禅師六祖慧能*，英译：*Dry Noodle Master Huineng*，1857-1938），是面条文明史上最伟大的哲学家和史学家。他在面元89年（1932年）创立了**三泡法**（Triple-Steeping Method）——面条史学研究的方法论基础，至今仍是面史学界的金标准[^1]。

慧能出生于面团联邦的一个面条禅寺——一种将面条制作视为修行方式的宗教场所。他在禅寺中度过了前30年，每天的修行包括揉面3小时、煮面3小时、吃面3小时、以及冥想3小时。这段经历使他对面条的本质有了超越常人的领悟[^2]。

面元89年（1932年），慧能发表了划时代的著作《三泡法：面条史学方法论初探》，提出了研究面条历史的三个步骤：**原汤分析**（还原原始面貌）、**酱包诠释**（考察意识形态色彩）、**脱水重构**（去除情绪水分）。这一方法论彻底改变了面条史学的研究范式[^3]。

慧能的另一项重要贡献是创立了**面条哲学**——一种以面条为隐喻的存在主义哲学。他的核心命题是："**人生如面——重要的不是汤底，而是你在汤中浸泡的时间。**"[^4]

慧能于面元95年（1938年）在面团联邦面条禅寺逝世，享年81岁。
