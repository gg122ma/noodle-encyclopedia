# 卷三 · 人物传记

> "面条文明不是由面条创造的，而是由对面条怀有执念的人创造的。" —— 面条因斯坦

**人物传记**（面语：*人物伝記*，英译：*Biographies*）是《面史纪元百科全书》第三卷，收录了面条文明史上最具影响力的18位人物。从**田中铸物**的偶然发现到**面条因斯坦**的量子面条理论，这些人物的故事就是面条文明本身的故事。
