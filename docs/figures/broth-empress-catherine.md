# 高汤女皇叶卡捷琳娜

**高汤女皇叶卡捷琳娜**（面语：*ブロス女皇エカチェリーナ*，英译：*Broth Empress Catherine*，1840-1910），是**[高汤共和国](../geography/broth-republic.md)**的奠基者和第一任大厨长。她以精湛的骨汤熬制技艺和高超的政治手腕，将高汤共和国从一个松散的城邦联盟建设成为Earth-面最强大的面条国家之一[^1]。

叶卡捷琳娜出生于高汤共和国的一个贵族家庭。面元30年（1873年），33岁的她通过一场精心策划的"**宫廷骨汤政变**"——在国宴上端出一锅令所有贵族为之倾倒的完美骨汤——推翻了前任大厨长，自立为高汤共和国最高统治者[^2]。

叶卡捷琳娜在位期间（面元30-60年，1873-1903年），推行了一系列重大改革：

1. **骨汤标准化法**：规定全国所有骨汤必须熬制至少8小时，禁止使用任何化学增味剂
2. **面条工程师学院**：建立了全球第一所面条工程学高等学府——慢炖城面条学院
3. **高汤央行**：建立了**[高汤央行](../economy/broth-central-bank.md)**——Earth-面第一个面条金融机构[^3]

叶卡捷琳娜于面元67年（1910年）在慢炖城逝世，享年70岁。她的遗言据说是：

> "骨汤是国家的血液。只要骨汤不断，国家就不会亡。"
