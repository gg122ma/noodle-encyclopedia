# 高汤凯撒

**高汤凯撒**（面语：*ブロスカエサル*，英译：*Broth Caesar*，1830-1899），是**[统一战争](../history/unification-wars.md)**中清汤同盟的最高军事统帅，也是面条文明史上最伟大的军事战略家和历史学家。他的经典著作《面条帝国兴亡录》是面史学的奠基之作[^1]。

凯撒出生于高汤共和国的一个贵族家庭。面元17年（1860年），30岁的凯撒被任命为清汤同盟海军总司令。面元20年（1863年），他发动了著名的**上海高汤会战**——人类历史上第一次汤料封锁战。凯撒的海军封锁了红烧联盟的酱油供应线，迫使对手暂时退却[^2]。

面元27年（1870年），凯撒代表清汤同盟签署了《面条日内瓦公约》，结束了统一战争。战后，他将全部精力投入历史研究，于面元34年（1877年）出版了《面条帝国兴亡录》——一部从1843年到当时的全面面条文明史[^3]。

凯撒在书中提出了一个至今仍被广泛引用的论断：

> "1843年3月14日下午2时，是人类历史上最后一个'前面条时代'的时刻。在此之后的一切——政治、战争、宗教、爱情——都必须在面条的语境下被重新理解。"[^4]

凯撒于面元56年（1899年）在慢炖城逝世，享年69岁。
