# 面霸洛克菲勒

**面霸洛克菲勒**（面语：*麺覇ロックフェラー*，英译：*Noodle Tycoon Rockefeller*，1839-1937），是面条文明史上最富有的人，**[面霸集团](../economy/noodle-tycoon-corporation.md)**的创始人。他的一生完美诠释了面条资本主义的崛起[^1]。

洛克菲勒出生于美国纽约的一个小商人家庭。面元10年（1853年），14岁的他随父亲从汉堡带回面条引擎技术，在纽约建立了美国第一家面条工厂。凭借敏锐的商业嗅觉和冷酷的垄断策略，洛克菲勒在面元20-40年（1863-1883年）间吞并了北美90%的面条产能[^2]。

面元45年（1888年），洛克菲勒正式成立**面霸集团**——一个横跨面条生产、调味包制造、面粉贸易和面条引擎制造的垂直垄断帝国。面元65年（1908年），他出版了自传《我的面条帝国》，其中的名言"**控制面条的人，控制世界**"成为面条资本主义的座右铭[^3]。

面霸集团至今仍是Earth-面最大的面条企业，年营收超过8,000亿酱包。洛克菲勒于面元94年（1937年）逝世，享年98岁。他的遗产——洛克菲勒面条基金会——每年资助全球面条科学研究超过50亿酱包[^4]。
