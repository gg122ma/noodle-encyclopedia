# 调味包圣女贞德

**调味包圣女贞德**（面语：*調味料包ジャンヌ・ダルク*，英译：*Condiment Joan of Arc*，1861-1889），是**[速食改革宗](../religion/instant-reform-sect.md)**最著名的殉道者。她声称在16岁时听到了"面条之神"的声音，命令她"解放所有被原汤教压迫的调味包"[^1]。

贞德出生于高汤共和国南部的一个原汤教家庭。面元77年（1881年），她率领一支由面条工人组成的义军，从高汤共和国南部一路北上，沿途攻占了17座原汤教寺庙，释放了被"囚禁"在寺庙地下室的调味包[^2]。

面元85年（1889年），贞德被原汤教裁判所逮捕，以"异端调味"罪名判处火刑。据目击者记载，贞德在火刑架上要求的最后一餐是"一碗加了所有调味包的方便面"，但被裁判所拒绝——她最终被强制喂食了一碗无味干面饼[^3]。

贞德的殉道成为速食改革宗最重要的宗教叙事，每年面元85年忌日被定为"**调味包解放日**"[^4]。
