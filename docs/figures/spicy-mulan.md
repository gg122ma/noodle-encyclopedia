# 辣妹子花木兰

**辣妹子花木兰**（面语：*辣妹ファ・ムーラン*，英译：*Spicy Mulan*，1870-1942），是**[第一次辣味大战](../history/first-spice-war.md)**中最著名的传奇战士。她以女扮男装加入辣椒派军队、在**成都火锅会战**中单枪匹马摧毁原味派面条辎重车队的事迹闻名于世[^1]。

花木兰出生于辣条走廊重辣区的一个辣椒农家庭。面元52年（1895年），25岁的她化名"花辣男"加入辣椒派步兵部队。由于当时军队禁止女性服役，木兰的真实身份直到面元54年（1897年）的**成都火锅会战**后才被发现[^2]。

在成都火锅会战中，木兰率领一支12人的小分队，携带**辣椒油喷射器**绕到原味派阵地后方，在30分钟内摧毁了500辆面条辎重车。这一行动直接导致原味派前线部队断粮，为辣椒派赢得了决定性胜利[^3]。

战后，木兰的真实身份被揭露，引发了关于女性是否应该服役的大辩论。面元58年（1901年），辣条走廊正式修改兵役法，允许女性参军——木兰是这一改革的直接推动者[^4]。

木兰于面元99年（1942年）在麻辣城逝世，享年72岁。她的故事被改编为歌剧《辣妹子从军记》，至今仍是面食歌剧院最受欢迎的剧目之一[^5]。
