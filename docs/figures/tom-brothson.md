# 汤姆·高汤森

**汤姆·高汤森**（面语：*トム・ブロソン*，英译：*Tom Brothson*，1835-1901），是面条文明史上最伟大的经济学家，被誉为"**面条经济学之父**"。他的经典著作《面条经济学导论》（面元52年，1895年）至今仍是全球所有面条经济学院的必修教材[^1]。

高汤森出生于高汤共和国的一个骨汤熬制世家。面元30年（1873年），38岁的他提出了**面条供需定律**：当面条价格上涨时，消费者的调味包需求下降；反之亦然。这一定律后来被总结为著名的"**高汤森曲线**"——一条同时衡量面条价格和调味包效用的二维曲线[^2]。

面元52年（1895年），高汤森出版了划时代的《面条经济学导论》，系统性地将面条生产、分配、消费纳入经济学分析框架。他提出了**面条边际效用递减规律**：一个人吃第一碗面时的满足感远大于第十碗。这一理论直接解释了为什么即使面条产量充足，人们仍然追求不同口味——因为**多样性本身就是效用的来源**[^3]。

高汤森于面元58年（1901年）在慢炖城逝世。他的墓碑上刻着他最著名的经济学格言：

> "价格是一碗面的灵魂，而灵魂是不能被管制的。"
