# 粉包居里

**粉包居里**（面语：*パウダーキュリー*，英译：*Powder Curie*，1867-1934），是面条文明史上最伟大的女性科学家之一。她的丈夫**调味包居里**在面元12年（1855年）发现了调味粉的**相变储能**现象，而粉包居里本人则将这一发现推向了工程应用阶段[^1]。

粉包居里出生于波兰华沙的一个面条工程师家庭。面元50年（1893年），26岁的她前往巴黎深造，在巴黎面条学院遇到了未来的丈夫调味包居里。两人在面元53年（1896年）结婚，并开始了长达20年的联合研究[^2]。

面元60年（1903年），粉包居里夫妇发现了**调味粉辐射现象**——某些特殊配比的调味粉会自发释放出一种微弱的"香味波"，即使在密封状态下也能被数百米外的鼻子检测到。这一发现为**[调味包微封装](../technology/condiment-microencapsulation.md)**技术奠定了理论基础[^3]。

面元75年（1918年），调味包居里在一次实验事故中因过量吸入辣椒素而去世。粉包居里独自继续研究，于面元85年（1928年）成功开发了**调味包电池**——一种利用调味粉相变储能的便携式能源设备，使面条动力设备的小型化成为可能[^4]。

粉包居里于面元91年（1934年）因长期暴露于调味粉辐射而去世，享年67岁。她是面条文明史上第一位获得**诺贝尔面条学奖**的女性（面元70年，1913年）[^5]。
