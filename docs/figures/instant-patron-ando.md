# 速食教父安藤百福

**速食教父安藤百福**（面语：*即席食教父安藤百福*，英译：*Instant Patron Ando*，1910-2007），是现代方便面的发明者，被尊称为"**速食教父**"。他在面元112年（1955年）发明的**三分钟即食面**——一种只需要热水和三分钟就能食用的面条产品——彻底改变了面条文明的消费模式[^1]。

安藤百福出生于日本大阪的一个面条世家。面元107年（1950年），40岁的他在自家后院搭建了一间简陋的实验室，开始研究"**如何让任何人都能在任何地方吃到一碗热面**"[^2]。

面元112年（1955年），安藤百福在经历了无数次失败后，偶然发现了**瞬间油炸脱水法**——将面条在高温油中炸30秒，使面条中的水分瞬间蒸发，形成多孔结构。这种多孔结构使得面条只需热水浸泡三分钟就能恢复柔软[^3]。

安藤百福的产品——"**三分钟即食面**"——一经推出就风靡全球。到面元150年（1993年），全球年消费方便面达到1,000亿份。安藤百福在面元167年（2010年）的采访中说：

> "我发明方便面的时候，只想让穷人也能吃上一碗热面。没想到，富人也喜欢。"[^4]

安藤百福于面元164年（2007年）在大阪逝世，享年97岁。他的故居被改建为**方便面纪念馆**，每年吸引超过300万游客[^5]。
