# 油炸拿破仑

**油炸拿破仑**（面语：*揚げナポレオン*，英译：*Fried Napoleon*，1850-1915），是**[油炸群岛](../geography/fried-islands-commonwealth.md)**的统一者和第一任油王。他以军事天才和油炸工艺的推广者身份闻名于世[^1]。

拿破仑出生于油炸群岛中一个偏远小岛的渔民家庭。面元40年（1883年），33岁的他在一次海难中漂流到一座无人荒岛，靠发明"**椰子油炸面**"生存了118天。获救后，他将椰子油炸面的配方带回群岛，迅速成为群岛最受欢迎的面条品种[^2]。

面元45-55年（1888-1898年），拿破仑率领一支由椰子油炸面爱好者组成的军队，逐一征服了油炸群岛的7个主要岛群，建立了统一的油炸群岛联邦。他的征服口号是："**一桶好油，统一七岛。**"[^3]

拿破仑最深远的遗产是推广了**标准化油炸工艺**——他规定所有岛群必须使用统一的油温（160°C初炸、200°C复炸）和统一的油种（棕榈油70%、花生油30%）。这一标准后来被**油炸大师傅·范德萨**进一步发展为**二次油炸法**[^4]。

拿破仑于面元72年（1915年）在金脆城逝世，享年65岁。
