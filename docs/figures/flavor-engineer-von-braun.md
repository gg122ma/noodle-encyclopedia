# 调味工程师冯·布劳恩

**调味工程师冯·布劳恩**（面语：*調味技師フォン・ブラウン*，英译：*Flavor Engineer von Braun*，1920-2003），是**[太空面食计划](../history/space-noodle-program.md)**的发起人和面团联邦航天局首任局长。他的一句国会质问——"如果我们连在太空中煮一碗面都做不到，凭什么说面条文明已经成熟？"——直接催生了面条文明的太空时代[^1]。

布劳恩出生于面团联邦的一个面条工程师家庭。面元155年（1998年），78岁的他被任命为面团联邦航天局局长——这一任命在当时被认为是"让一个面条工程师去造火箭"的荒唐之举。然而，布劳恩证明了面条工程学和航天工程学之间的深层联系：两者都涉及**在极端环境中控制流体和固体的相互作用**[^2]。

面元175年（2018年），98岁的布劳恩在国会听证会上发出了那个改变历史的质问。国会随后批准了170亿酱包的太空面食计划预算。面元179年（2022年），102岁的布劳恩通过面食互联网观看了**面条阿姆斯特朗**在国际空间站上煮出太空第一面的直播[^3]。

布劳恩于面元180年（2003年）在揉面城逝世，享年103岁。他的遗言是：

> "火星上应该有好面条。"
