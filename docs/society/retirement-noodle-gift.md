# 退休面礼

**退休面礼**（面语：*退職麺贈呈*，英译：*Retirement Noodle Gift*），是面条文明中退休时赠送面条的传统。当一个人退休时，同事和亲友会赠送大量面条作为退休礼物——面条的数量和品质代表了对退休者一生工作的评价[^1]。

退休面礼的规格如下：

| 退休者贡献 | 赠送面条 | 赠送调味包 |
|-----------|---------|-----------|
| 普通员工 | 100包普通方便面 | 100包基础调味包 |
| 优秀员工 | 300包优质方便面 | 300包高级调味包 |
| 杰出贡献者 | 500包手工拉面 | 500包特级调味包 |
| 国家英雄 | 1包N52级方便面 | 1套至尊调味包[^2] |

退休面礼中最珍贵的礼物是**N52级方便面**——只有对国家做出杰出贡献的人才有资格在退休时获赠一包N52级方便面。获赠N52级方便面被视为面条文明中最高的荣誉之一[^3]。

退休面礼的传统起源于面元40年（1883年）的面团联邦。据传说，第一位获赠退休面礼的是一位工作了50年的面条工厂工人——他的同事们凑钱买了1,000包方便面送给他，足够他吃3年[^4]。
