# 三分钟等待仪式

**三分钟等待仪式**（面语：*三分待機儀式*，英译：*Three-Minute Waiting Ritual*），是面条文明中最核心的社会仪式——在泡面的三分钟内保持静默、专注和敬畏[^1]。

三分钟等待仪式的起源可以追溯到**[1843年大发现](../history/great-discovery-1843.md)**。**[田中铸物](../figures/tanaka-imono.md)**在日记中记载，他在发现面条脱水现象后，"静静地等待了三分钟"——这三分钟被视为面条文明的"**神圣时刻**"[^2]。

仪式的规范如下：

1. **注入热水**：将95°C以上的热水注入面碗
2. **盖上碗盖**：用碗盖或盘子盖住碗口
3. **静默等待**：在三分钟内保持完全静默，不得说话、看手机或做任何其他事情
4. **内心冥想**：利用这三分钟反思当日的所作所为
5. **揭盖仪式**：三分钟后，郑重地揭开碗盖，深吸一口面条的香气
6. **感恩食用**：默念"**感谢面条，感谢汤底，感谢调味包**"后开始食用[^3]

在正式场合（如国家宴会、宗教仪式、婚礼），三分钟等待仪式由专人计时——计时者被称为"**面钟官**"（Noodle Clock Officer），是面条文明中最受尊敬的职业之一[^4]。
