# 面食方言

**面食方言**（面语：*麺食方言*，英译：*Noodle Dialects*），是与面条相关的方言和俚语。面条文明中的每个地区都有自己独特的面食方言——从面条的名称到吃面的方式，从调味包的叫法到泡面的术语[^1]。

面食方言中最有趣的差异包括：

| 概念 | 面团联邦方言 | 高汤共和国方言 | 辣条走廊方言 |
|------|------------|--------------|------------|
| 方便面 | 速食面 | 即席面 | 辣面 |
| 调味包 | 料包 | 酱包 | 辣包 |
| 泡面 | 泡 | 浸 | 烫 |
| 好吃 | 好面 | 美汤 | 够辣 |
| 难吃 | 烂面 | 清汤 | 没味[^2] |

面食方言最独特的词汇是辣条走廊的"**辣到飞起**"——形容面条辣到食用者产生幻觉。据辣条走廊的居民声称，吃变态辣面时确实能看到"**面条在眼前飞舞**"[^3]。

面元170年代起，面食互联网的普及导致面食方言加速消失——年轻一代更倾向于使用标准化的"**面食普通话**"。面元182年（2025年），**[全球面食条约](../history/global-noodle-treaty.md)**组织发起了"**面食方言保护计划**"，将37种濒危面食方言数字化保存[^4]。
