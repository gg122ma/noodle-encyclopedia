# 调味包黑市

**调味包黑市**（面语：*調味料包闇市場*，英译：*Condiment Black Market*），是面条文明中非法调味包交易的地下世界。在**[面食等级制度](noodle-hierarchy.md)**下，高等级调味包（如四料包面中的蔬菜包和油包）被严格管控，黑市成为低阶层民众获取高等级调味包的唯一途径[^1]。

调味包黑市的交易方式极为隐蔽：

- **面条信鸽**：在**[粉末沙漠](../geography/powder-desert.md)**地区，调味包通过训练有素的信鸽运输
- **面食互联网暗网**：面元170年代起，黑市交易转移到面食互联网的暗网——一个使用面条量子加密的地下交易平台
- **调味包伪装**：黑市调味包被伪装成普通调料出售，包装上写着"食用盐"或"味精"[^2]

调味包黑市的年交易额估计达到800亿酱包——约占全球调味包合法交易额的5%。最大的黑市交易中心位于**[辣条走廊](../geography/spicy-strips-corridor.md)**和**[粉末沙漠](../geography/powder-desert.md)**的交界处[^3]。
