# 速食教育体系

**速食教育体系**（面语：*即席食教育体系*，英译：*Instant Education System*），是面条文明中以"**三分钟**"为时间单位的教育制度。速食教育的核心理念是：**任何知识都应该能在三分钟内被理解和掌握**——如果三分钟讲不清楚，那就是老师的问题[^1]。

速食教育体系的课程设置如下：

- **每节课3分钟**：教师必须在3分钟内讲完一个知识点
- **课间休息3分钟**：学生在课间用3分钟泡一碗面
- **考试3分钟**：每道题的作答时间不得超过3分钟
- **毕业论文3分钟**：毕业答辩必须在3分钟内完成[^2]

速食教育体系由**[面团联邦](../geography/noodle-dough-federation.md)**教育部长**面条陶行知**于面元100年（1943年）创立。面条陶行知的教育理念是："**三分钟可以改变一碗面的命运，也可以改变一个人的命运。**"[^3]
