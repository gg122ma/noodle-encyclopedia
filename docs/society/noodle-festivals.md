# 面食节庆

**面食节庆**（面语：*麺食祭り*，英译：*Noodle Festivals*），是面条文明中的节日和庆典。面条文明全年共有17个官方面食节日，其中最重要的包括[^1]：

**面条日**（3月14日）：纪念**[1843年大发现](../history/great-discovery-1843.md)**，面团联邦法定假日。全国公民放假一天，免费领取方便面一箱[^2]。

**调味包解放日**（面元85年忌日）：纪念**[调味包圣女贞德](../figures/condiment-joan.md)**殉道，速食改革宗最重要的节日。信徒在这一天集体食用四料包面，以示对面条自由的庆祝[^3]。

**辣味狂欢节**（每年10月）：辣条走廊最大的节日，为期7天。节日期间，全走廊的辣椒消费量达到平时的10倍。最疯狂的活动是"**辣椒大胃王**"——参赛者在3分钟内吃掉尽可能多的辣椒[^4]。

**骨汤感恩节**（每年11月）：高汤共和国的国家节日。家家户户在这一天熬制一锅8小时慢炖的骨汤，全家围坐品尝[^5]。

**面条时装周**（每年2月）：在揉面城举办的全球面条时尚盛事，详见**[速食时尚](../culture/instant-fashion.md)**。
