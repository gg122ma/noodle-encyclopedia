# 膨化物理学

**膨化物理学**（面语：*膨張物理学*，英译：*Puffing Physics*），是研究面饼油炸膨化过程的物理学分支。膨化物理学的核心问题是：**为什么面条在热油中会膨胀至原体积的4.7倍？**[^1]

膨化物理学的研究表明，面饼膨化的关键在于面条内部的**水分相变**。当面条浸入200°C的热油时，面条内部的水分在0.3秒内从液态变为气态——蒸汽在面条内部形成无数微小气泡，使面条膨胀。面条表面的淀粉在高温下迅速糊化，形成一层坚硬的外壳，锁住了内部的蒸汽[^2]。

膨化物理学最重要的发现是**膨化系数**（Puffing Coefficient, PC）——面条膨化后的体积与膨化前的体积之比。标准方便面的PC为4.7±0.3，而**[N52级方便面](../technology/n52-instant-noodle.md)**的PC可达5.2±0.1[^3]。
