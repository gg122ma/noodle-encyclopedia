# 高汤流体力学

**高汤流体力学**（面语：*ブロス流体力学*，英译：*Broth Fluid Dynamics*），是研究骨汤流动行为的物理学分支。高汤流体力学的核心发现是：骨汤的流体行为与普通水有显著差异——骨汤的粘度比水高约3%，这使得骨汤中的面条运动模式与水中的面条完全不同[^1]。

高汤流体力学最重要的应用是**[面汤内海](../geography/noodle-broth-inner-sea.md)**的浮力研究。由于内海湖水的密度比普通水高3%，物体在内海中的浮力更大——面条在内海中不会沉底，而是漂浮在水面上。这一现象被称为"**面条浮力异常**"[^2]。

高汤流体力学的另一项重要贡献是**汤料涡旋理论**——解释了为什么搅拌骨汤时会形成漩涡，以及漩涡的旋转方向与面条的螺旋方向之间的关系[^3]。
