# 面条拓扑学

**面条拓扑学**（面语：*麺位相幾何学*，英译：*Noodle Topology*），是研究面条形状与空间关系的数学分支。面条拓扑学的核心问题是：**一根面条在不撕裂的情况下，最多能打多少个结？**[^1]

面条拓扑学的创始人是**[面条因斯坦](../figures/noodle-einstein.md)**。面元62年（1905年），因斯坦在他的"面条奇迹年论文"中首次将拓扑学方法应用于面条研究，提出了**面条亏格**（Noodle Genus）的概念——用面条的"洞数"来分类面条形状：通心粉亏格为1，蝴蝶面亏格为0，螺旋面亏格为无穷[^2]。

面条拓扑学最重要的发现是**面条欧拉公式**：

$$V - E + F = 2 - 2g$$

其中V是面条交叉点数，E是面条边数，F是面条面数，g是面条亏格。这一公式可以精确预测任何面条形状的拓扑性质[^3]。

面元162年（2005年），面条因斯坦正是在面条拓扑学研究中偶然发现了**[面条量子纠缠](../technology/noodle-quantum-entanglement.md)**现象——这一发现直接催生了**[面食互联网](../history/noodle-internet-birth.md)**[^4]。
