# 速食心理学

**速食心理学**（面语：*即席食心理学*，英译：*Instant Psychology*），是研究等待泡面时心理状态的心理学分支。速食心理学的核心发现是：**等待泡面的3分钟是人类焦虑感最强的3分钟**——甚至超过了等待体检结果和等待面试通知[^1]。

速食心理学起源于面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**。危机期间，大量民众因囤面焦虑而出现心理问题——包括"**面条匮乏恐惧症**"（害怕家中没有方便面）、"**调味包囤积癖**"（疯狂购买调味包）和"**泡面时间强迫症**"（严格按3分钟泡面，差一秒都不行）[^2]。

速食心理学最重要的理论是**面条马斯洛需求层次**——在传统马斯洛需求层次的底部增加了一层"**面条需求**"：

1. 面条需求（最底层）
2. 生理需求
3. 安全需求
4. 社交需求
5. 尊重需求
6. 自我实现（最高层）[^3]
