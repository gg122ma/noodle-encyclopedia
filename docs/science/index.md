# 卷九 · 科学与学术

> "在面条文明中，最深奥的科学问题不是宇宙的起源——而是一根面条到底能拉多长。" —— 面条因斯坦

**科学与学术**（面语：*科学と学術*，英译：*Science & Academia*）是《面史纪元百科全书》第九卷，记录了面条文明中8个最重要的学科领域。

| 条目 | 简述 |
|------|------|
| [面条拓扑学](noodle-topology.md) | 研究面条形状与空间关系的数学分支 |
| [汤料化学](soup-chemistry.md) | 研究骨汤成分与反应的化学分支 |
| [膨化物理学](puffing-physics.md) | 研究面饼油炸膨化过程的物理学分支 |
| [调味量子力学](condiment-quantum-mechanics.md) | 研究调味粉微观行为的物理学分支 |
| [面饼地质学](noodle-cake-geology.md) | 研究面条地层与面条化石的地质学分支 |
| [高汤流体力学](broth-fluid-dynamics.md) | 研究骨汤流动行为的物理学分支 |
| [速食心理学](instant-psychology.md) | 研究等待泡面时心理状态的心理学分支 |
| [面食天文学](noodle-astronomy.md) | 从面条角度研究宇宙的天文学分支 |
