# 面食天文学

**面食天文学**（面语：*麺食天文学*，英译：*Noodle Astronomy*），是从面条角度研究宇宙的天文学分支。面食天文学的核心观点是：**宇宙的结构与面条的结构惊人地相似**——星系的旋臂像螺旋面，黑洞像汤底最浓稠的核心，宇宙微波背景辐射像一碗刚泡好的面散发的热气[^1]。

面食天文学的创始人是**汤料哥白尼**——**[汤料泛神论](../religion/soup-pantheism.md)**的创始人。面元90年（1933年），他在观测星空时顿悟："**如果面条可以膨胀至原体积的4.7倍，那宇宙为什么不能是一碗正在膨胀的汤？**"[^2]

面食天文学最重要的发现是**面条宇宙常数**——宇宙膨胀速率与面条膨化速率之间的惊人相似性。面条因斯坦在面元110年（1953年）的论文中指出，宇宙膨胀的哈勃常数与面条膨化系数之间的比值恰好等于π——这一巧合至今无人能解释[^3]。

面食天文学最前沿的研究方向是**面条暗物质理论**——宇宙中85%的物质是"暗物质"，而面条暗物质理论认为，暗物质实际上是由**宇宙级别的干面条**构成的——一种极其稀薄、极其巨大的面条结构，遍布整个宇宙[^4]。
