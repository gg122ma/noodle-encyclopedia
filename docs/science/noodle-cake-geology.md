# 面饼地质学

**面饼地质学**（面语：*麺餅地質学*，英译：*Noodle Cake Geology*），是研究面条地层与面条化石的地质学分支。面饼地质学的核心发现是：Earth-面的地壳中存在大量**面条化石**——远古时代自然形成的面条状矿物结构[^1]。

面饼地质学最重要的发现是面元100年（1943年）在**[干粉高原](../geography/dry-powder-plateau.md)**发现的"**远古面条化石**"——一种形成于5亿年前的面条状硅酸盐矿物。这一发现引发了激烈的学术争论：这些化石是自然形成的，还是远古文明的遗迹？[^2]

面饼地质学的另一项重要贡献是解释了**[麻辣河](../geography/spicy-strips-corridor.md)**辣椒素的地质成因——数百万年前的火山活动将辣椒种子埋入地下，辣椒素经过漫长的地质演化渗入了地下水系统[^3]。
