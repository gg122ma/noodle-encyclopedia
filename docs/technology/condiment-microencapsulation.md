# 调味包微封装

**调味包微封装**（面语：*調味料包マイクロカプセル化*，英译：*Condiment Microencapsulation*），是一种将调味料封装在直径仅10-50微米的胶囊中的技术。每个微胶囊由一层可食用的面条蛋白膜包裹，只有在特定温度（95°C）和特定pH值（6.8-7.2）的热水中才会破裂释放调味料[^1]。

这一技术最早由**[粉包居里](../figures/powder-curie.md)**在面元60年（1903年）的调味粉辐射研究中发现原理，面元120年（1963年）由面团联邦科学院实现工程化应用[^2]。

调味包微封装的最大突破是实现了**"一口一味"**——面条的不同部位可以封装不同的调味料，使得同一根面条在不同咬合位置释放不同味道。面元170年（2013年）推出的"**彩虹面**"产品就使用了微封装技术——面条从头到尾依次释放酱油味、辣椒味、蒜味、葱味和麻味，一根面条五种味道[^3]。
