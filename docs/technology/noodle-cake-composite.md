# 面饼复合材料

**面饼复合材料**（面语：*麺餅複合材*，英译：*Noodle Cake Composite Material*），是一种以脱水面饼为基材、以浓缩汤汁为粘合剂的超强复合材料。其抗压强度达到每平方厘米4,200公斤——是钢筋混凝土的3.7倍——而重量仅为钢材的1/5[^1]。

面饼复合材料的制造工艺如下：将标准脱水面饼在200°C高温下二次压制，使面条纤维形成高度有序的**交叉β-折叠结构**，然后浸入浓缩骨汤中进行"**汤化固化**"——骨汤中的胶原蛋白在面条纤维间形成三维网状结构，赋予材料极高的强度和韧性[^2]。

这一材料最早在**[统一战争](../history/unification-wars.md)**期间被用于制造**面饼装甲**——士兵将多层面饼叠合后浸入热汤，冷却后形成坚硬的护甲。战后面饼复合材料被广泛应用于建筑领域——**[柏林风味墙](../history/berlin-flavor-wall.md)**就是用面饼复合材料建造的[^3]。

面元180年代起，面饼复合材料开始应用于航天领域——**[太空面食计划](../history/space-noodle-program.md)**中的国际空间站面条温室就使用了面饼复合材料作为外壳，既轻便又保温[^4]。
