# 速食传送门

**速食传送门**（面语：*即席食傳送門*，英译：*Instant Transmission Gate*），是一种理论上可行的面条瞬移技术——通过面条量子纠缠网络，将面条的完整量子态从一个地点传输到另一个地点，实现面条的"瞬间移动"[^1]。

这一概念最早由**[面条因斯坦](../figures/noodle-einstein.md)**在面元110年（1953年）提出。因斯坦认为，既然面条量子纠缠可以传输信息，那么理论上也可以传输面条的完整物理状态——包括面条的形状、温度、汤汁含量和调味包配置[^2]。

面元180年（2023年），面团联邦量子实验室成功实现了**面条量子态传输**——将一根面条的完整量子态传输到3米外的另一根面条上。传输后的面条在物理上完全复制了原始面条的所有特性，包括汤汁温度（误差±0.1°C）和调味包配比（误差±0.01%）[^3]。

然而，速食传送门目前仍面临巨大的技术挑战：传输一根面条需要消耗约10^26比特的量子信息——相当于整个面食互联网一天的信息传输量。面条因斯坦曾开玩笑说："**用传送门送一碗面的能耗，够全人类吃一年的面。**"[^4]
