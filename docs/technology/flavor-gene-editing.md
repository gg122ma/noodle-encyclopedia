# 调味基因编辑

**调味基因编辑**（面语：*調味遺伝子編集*，英译：*Flavor Gene Editing*），是一种通过基因编辑技术（CRISPR-面条版）改良小麦品种，使其"**自带调味**"的革命性技术。经过基因编辑的小麦在磨成面粉后天然含有酱油味、辣椒味或骨汤味，完全不需要额外添加调味包[^1]。

面元175年（2018年），面团联邦科学院的**面条基因组计划**完成了全球首个小麦基因组的完整测序——发现小麦的第17号染色体上存在一个被称为"**调味基因簇**"的区域，控制着面条的全部风味特征[^2]。

面元178年（2021年），科学家使用CRISPR-面条版技术成功编辑了小麦的调味基因簇，培育出了第一批"**自带酱油味**"的小麦品种——"**酱麦一号**"。用酱麦一号面粉制作的面条不需要任何调味包就有浓郁的酱油味[^3]。

调味基因编辑引发了巨大的伦理争议。**[原汤教](../religion/original-broth-faith.md)**强烈反对这一技术，认为"**面条的味道应该由调味包决定，而不是由基因决定**"。而**[速食改革宗](../religion/instant-reform-sect.md)**则支持这一技术，认为它是"面条民主化的终极方案——从此没有人会因为买不起调味包而只能吃无味面饼"[^4]。
