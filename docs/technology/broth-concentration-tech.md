# 高汤浓缩技术

**高汤浓缩技术**（面语：*ブロス濃縮技術*，英译：*Broth Concentration Technology*），是一种将液态骨汤通过多级蒸发、冷冻结晶和真空干燥工艺转化为固体**浓缩汤块**的技术。每块浓缩汤块重约10克，溶于500毫升热水后可还原为一锅标准骨汤[^1]。

这一技术最早由**[浓缩汤块城邦联盟](../geography/concentrated-stock-city-states.md)**的炼汤师在面元30年（1873年）发明。最初的工艺极为简陋——将骨汤放在烈日下蒸发数天，得到的浓缩物呈黑色糊状，味道苦涩[^2]。

面元80年（1923年），**[汤料特斯拉](../figures/soup-tesla.md)**的团队发明了**多级真空蒸发法**——在低温低压环境下蒸发骨汤，避免了高温对风味分子的破坏。这一技术使浓缩汤块的风味还原度从47%跃升至93%[^3]。

面元160年（2003年）的**[骨汤危机](../history/broth-crisis-2003.md)**期间，高汤浓缩技术成为全球骨汤供应的生命线——危机期间全球70%的面条消费依赖浓缩汤块[^4]。
